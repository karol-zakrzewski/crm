# Force CRM

Force CRM to aplikacja do zarządzania relacjami z klientami. W aplikacji można dodawać firmy, wyświetlać je

## Installation

Use the package manager npm to install Force CRM app.

```bash
npm install
```

Run application

```bash
npm run dev
```
